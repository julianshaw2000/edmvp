# AccuTrac Configuration Reference

## Environment Overview

| Service | Local | Production |
|---|---|---|
| Angular SPA | http://localhost:4200 | https://accutrac.org |
| .NET API | http://localhost:5000 | https://api.accutrac.org |
| PostgreSQL | Neon (pooler) | Neon (pooler) |
| File Storage | Local disk | Cloudflare R2 |
| Auth | Auth0 | Auth0 |
| Email | Log to console | SendGrid |

## .NET API Configuration

File: `packages/api/src/Tungsten.Api/appsettings.json` (defaults)
Override: `appsettings.Development.json` (local) or environment variables (Render)

### Connection Strings

```json
"ConnectionStrings": {
  "DefaultConnection": "Host=<host>;Database=<db>;Username=<user>;Password=<pass>;SSL Mode=VerifyFull"
}
```

| Env Var | Description |
|---|---|
| `ConnectionStrings__DefaultConnection` | Neon PostgreSQL connection string with pooler endpoint |

Source: `docs/neon.secrets`

### Auth0

```json
"Auth0": {
  "Domain": "<tenant>.auth0.com",
  "Audience": "https://api.accutrac.org"
}
```

| Env Var | Description |
|---|---|
| `Auth0__Domain` | Auth0 tenant domain (e.g., `dev-abc123.auth0.com`) |
| `Auth0__Audience` | Auth0 API identifier: `https://api.accutrac.org` |

Setup guide: `docs/auth0-setup.md`

### Cloudflare R2

```json
"R2": {
  "AccountId": "5418040d79f34e5e0b21cd4b6389adde",
  "AccessKeyId": "<r2-access-key>",
  "SecretAccessKey": "<r2-secret-key>",
  "BucketName": "accutrac",
  "PublicUrl": ""
}
```

| Env Var | Description |
|---|---|
| `R2__AccountId` | Cloudflare account ID: `5418040d79f34e5e0b21cd4b6389adde` |
| `R2__AccessKeyId` | R2 API token access key |
| `R2__SecretAccessKey` | R2 API token secret key |
| `R2__BucketName` | R2 bucket name: `accutrac` |
| `R2__PublicUrl` | Public URL for the bucket (if using custom domain) |

Source: `docs/cloudfare.secrets`

**Behavior:** If `R2:AccountId` is empty, falls back to `LocalFileStorageService` (stores to `./storage/` on disk).

### App

```json
"App": {
  "BaseUrl": "https://accutrac.org"
}
```

| Env Var | Description |
|---|---|
| `App__BaseUrl` | Base URL for QR codes on Material Passports and share links |

### Sentry

```json
"Sentry": {
  "Dsn": ""
}
```

| Env Var | Description |
|---|---|
| `Sentry__Dsn` | Sentry DSN for error tracking (optional, leave empty to disable) |

### CORS

```json
"Cors": {
  "AllowedOrigins": ["http://localhost:4200"]
}
```

| Env Var | Description |
|---|---|
| `Cors__AllowedOrigins__0` | Allowed origin for CORS. Default: `http://localhost:4200`. Set to `https://accutrac.org` in production. |

## Angular Configuration

File: `packages/web/src/environments/environment.ts` (dev)
File: `packages/web/src/environments/environment.production.ts` (prod)

### Development

```typescript
export const environment = {
  production: false,
  apiUrl: 'http://localhost:5000',
  auth0: {
    domain: 'YOUR_AUTH0_DOMAIN',
    clientId: 'YOUR_AUTH0_CLIENT_ID',
    audience: 'https://api.accutrac.org',
  },
};
```

### Production

```typescript
export const environment = {
  production: true,
  apiUrl: 'https://api.accutrac.org',
  auth0: {
    domain: 'YOUR_AUTH0_DOMAIN',
    clientId: 'YOUR_AUTH0_CLIENT_ID',
    audience: 'https://api.accutrac.org',
  },
};
```

## Render Deployment Configuration

### API (Web Service)

| Setting | Value |
|---|---|
| Name | `accutrac-api` |
| Runtime | Docker or .NET |
| Build Command | `cd packages/api && dotnet publish src/Tungsten.Api -c Release -o out` |
| Start Command | `cd packages/api/out && dotnet Tungsten.Api.dll` |
| Health Check Path | `/health` |

**Environment Variables:** Set all `ConnectionStrings__DefaultConnection`, `Auth0__*`, `R2__*`, `App__BaseUrl`, `Cors__AllowedOrigins__0` as above.

### Angular (Static Site)

| Setting | Value |
|---|---|
| Name | `accutrac-web` |
| Build Command | `cd packages/web && npm install && ng build --configuration production` |
| Publish Directory | `packages/web/dist/web/browser` |

### Custom Domain

| Domain | Service |
|---|---|
| `accutrac.org` | Static Site (Angular) |
| `api.accutrac.org` | Web Service (.NET API) |

Add both as custom domains in Render, then set DNS:
- `accutrac.org` → CNAME to Render static site URL
- `api.accutrac.org` → CNAME to Render web service URL

## Running Locally

```bash
# Terminal 1: API
cd packages/api/src/Tungsten.Api
dotnet run

# Terminal 2: Angular
cd packages/web
ng serve

# Open http://localhost:4200
```

The API auto-runs migrations and seeds reference data on startup in Development mode.
